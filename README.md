# Timeline Kit (`com.air.unity-timeline-kit`)

Unity Timeline 扩展：自定义 Clip/Track/Mixer、引用导出、`ITimelineAssetLoader`。

**分层：** 域包，仅依赖 `com.unity.timeline`；不依赖 `com.air.unity-game-core`。

## 安装

```json
"com.air.unity-timeline-kit": "file:../CustomPackages/packages/com.air.unity-timeline-kit"
```

Unity 2021.3+。
